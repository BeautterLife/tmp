# 범죄, 멈춰!

## 1. 프로젝트 소개


본 프로젝트는 실시간 위험행동 감지 CCTV 이다. CCTV 영상에서 행동 분류(action classification)를 하는 AI 기술을 사용하여 위험 행동인지를 판단한다. 위험 행동으로 감지된 결과는 관리자에게 알린다. 
위험 상황을 실시간으로 대응할 수 있고, CCTV 관리 및 인력 비용 절감 효과가 기대된다.
